# MZ Licitações

Sistema de Gestão de Licitações.

## Tecnologias

Este projeto foi construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Como rodar localmente

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Rode o servidor de desenvolvimento: `npm run dev`

## Deploy

Para fazer deploy, configure sua hospedagem favorita (Vercel, Netlify, etc) e conecte ao repositório.
